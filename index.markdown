## Instructions

To add your own PDF:

1. Place your PDF file in the `assets/pdfs/` directory
2. Update the `src` attribute in the embed tag above to match your filename
3. Example: `<embed src="/assets/pdfs/resume.pdf" type="application/pdf" width="100%" height="600px" />`

## Alternative: Link to PDF

If you prefer to link to the PDF instead of embedding it:

```html
<a href="/assets/pdfs/your-file.pdf" target="_blank">Open PDF</a>
```
